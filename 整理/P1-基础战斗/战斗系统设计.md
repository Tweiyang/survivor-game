## 🔫 自动射击流程

```mermaid
sequenceDiagram
    participant AA as AutoAttackComponent
    participant EM as EntityManager
    participant PF as ProjectileFactory
    participant PC as ProjectileComponent
    participant CS as CombatSystem
    participant HC as HealthComponent

    loop 每帧
        AA->>EM: findAllByTag('enemy')
        EM-->>AA: 敌人列表
        AA->>AA: 计算最近敌人
        alt cooldown 结束
            AA->>PF: 创建投射物(方向, 伤害)
            PF-->>PC: 新投射物实体
        end
    end

    loop 投射物每帧
        PC->>PC: 移动(speed × deltaTime)
        PC->>PC: 碰撞检测(enemyTags)
        alt 命中
            PC->>CS: dealDamage(owner, target, damage)
            CS->>HC: takeDamage(amount)
            alt HP <= 0
                HC->>HC: die() → emit('onDeath')
            end
            PC->>EM: remove(投射物)
        end
    end
```

## 🧟 怪物 AI：ChaseAI

```mermaid
flowchart TD
    A[每帧 update] --> B[查找最近玩家]
    B --> C{距离 < 攻击范围?}
    C -->|是| D[接触伤害 dealDamage]
    C -->|否| E[朝玩家方向移动]
    E --> F[speed × deltaTime]
```

### 数据驱动配置

```json
{
  "slime": {
    "name": "史莱姆",
    "stats": { "maxHp": 30, "attack": 5, "moveSpeed": 60 },
    "expValue": 10
  }
}
```

**Unity 对应**：`ScriptableObject` 怪物模板

## 💎 掉落与经验

```mermaid
flowchart LR
    A[怪物死亡] --> B[DropItemFactory.spawnDrops]
    B --> C[生成 N 个经验球]
    C --> D[DropComponent 磁吸飞向玩家]
    D --> E[碰到玩家 → onPickup 事件]
    E --> F[ExperienceSystem 加经验]
    F --> G{exp >= expToNextLevel?}
    G -->|是| H[升级! emit onLevelUp]
    G -->|否| I[继续积累]
```

### 升级公式

```
expToNextLevel = baseExp × multiplier^(level - 1)
                = 50 × 1.3^(level - 1)

Lv.1: 50   Lv.2: 65   Lv.3: 85   Lv.5: 143
```

## ⚡ 设计技巧

| 技巧 | 说明 |
|------|------|
| **统一伤害入口** | 所有伤害都走 `CombatSystem.dealDamage()`，方便加护甲/暴击/减伤 |
| **工厂模式** | `MonsterFactory.create(type)` 从 JSON 读配置自动组装组件 |
| **磁吸拾取** | 经验球不需要精确走到，进入范围自动飞向玩家，手感好 |
| **公式外置** | 升级经验、伤害系数全在 `formulas.json`，改数值不改代码 |
